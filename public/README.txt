Matheus & Isadora — final logo exports

Format:
- PNG-24 with transparency
- ICO favicons included
- SVG was not generated in this package because the current workflow was raster-based and forcing vectorization would risk fidelity loss.

Included variants:
- gold: premium antique gold for light/cream backgrounds
- charcoal: dark version for light backgrounds
- ivory: warm light version for dark/charcoal backgrounds

Important note:
This monogram is near-square. The horizontal header exports (400x100, 350x75, 250x70) are valid assets, but they naturally carry more side breathing room than a true horizontal wordmark. If you later want a stronger header lockup, the correct move is to pair the monogram with the names, not to distort the mark.
